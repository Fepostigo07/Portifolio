# <DevPortfolio />

Bem-vindo ao repositório do meu portfólio pessoal! Este projeto foi construído para apresentar minha trajetória como estudante de Engenharia de Software na PUCPR, destacando minhas habilidades e os projetos que desenvolvi ao longo do meu aprendizado.

Atualmente, estou em busca de uma oportunidade de estágio profissional onde eu possa aplicar meus conhecimentos e continuar crescendo na área de tecnologia.

## 💻 Tecnologias Utilizadas no Portfólio

Este site foi desenvolvido com foco em responsividade e design limpo, utilizando:

* **HTML5 & CSS3:** Estruturação e estilização da página.
* **JavaScript:** Lógica para interações dinâmicas, como o alternador de temas.
* **Bootstrap 5.3:** Framework CSS utilizado para garantir a adaptação a diferentes tamanhos de tela (mobile-first) e agilizar o desenvolvimento da interface.

## 🚀 Minhas Habilidades

Tenho interesse e atuo no desenvolvimento front-end, back-end e full-stack. Minhas principais ferramentas incluem:

* **Front-end:** HTML, CSS, JavaScript e Bootstrap.
* **Back-end & Banco de Dados:** Python, PHP, Java (com foco em Programação Orientada a Objetos), MySQL e SQL.
* **Versionamento:** Git e GitHub.

## ✨ Funcionalidades do Site

* **Design Responsivo:** A interface se adapta perfeitamente a celulares, tablets e desktops.
* **Dark/Light Mode:** Alternador de tema nativo desenvolvido com JavaScript e as variáveis de cor do Bootstrap, com salvamento da preferência do usuário (LocalStorage).
* **Navegação Suave:** Menu fixo com *ScrollSpy* (destaca a seção atual no menu) e rolagem suave ao clicar nos links.
* **Hospedagem:** Disponível online e de forma gratuita através do GitHub Pages.
